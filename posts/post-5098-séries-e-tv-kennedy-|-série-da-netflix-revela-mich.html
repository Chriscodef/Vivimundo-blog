Créditos da imagem: Netflix/Reprodução A Netflix revelou a primeira foto de Kennedy , série que vai contar a história de uma das famílias mais poderosas da política dos Estados Unidos. A imagem traz Michael Fassbender ( X-Men: Primeira Classe ) como Joe Kennedy Sr. , o patriarca do clã. A série é baseada no livro JFK: Coming of Age in the American Century, 1917–1956 , de Fredrik Logevall , e conta como Joe Sr., a esposa Rose ( Laura Donnelly ) e os nove filhos ascenderam à elite da nação. Confira abaixo a primeira foto de Michael Fassbender na produção: A Netflix também anunciou o elenco da produção, que conta com Nick Robinson ( Jurassic World ) como Joe Kennedy Jr., Joshuah Melnick ( Não Alimente As Crianças ) como Jack Kennedy, Ben Miles ( Andor ) como Eddie Moore, Lydia Peckham ( Cowboy Bebop ) como Rosemary Kennedy e Saura Lightfoot-Leon ( Mestres do Ar ) como Kick Kennedy. Além disso, a produção terá Wyatt Russell ( Thunderbolts* ) como Charles Lindbergh, Imogen Poots ( Sala Verde ) como Gloria Swanson, Cole Doman ( The Mastermind ) como Lem Billings, Georgina Bitmead ( Heartstopper ) como Eunice Kennedy, Patrick Fischler ( Paradise ) como Arthur Krock, Caitlin FitzGerald ( Succession ) como Clare Boothe Luce, Albert Welling ( Succession ) como Winston Churchill, Toby Huss ( A Hora do Mal ) como Franklin D. Roosevelt e Eddie Marsan ( Heartstopper ) como J. Edgar Hoover. Kennedy terá Sam Shaw ( Castle Rock ; Masters of Sex ) como showrunner e Thomas Vinterberg ( Druk: Mais uma Rodada ) como diretor.
