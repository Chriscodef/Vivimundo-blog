Software Opera Neon vira " central" para Chat GPT, Claude e outros agentes Há 2 horas

Por Kevin Ribeiro • 31/03/2026 às 11:19

A Opera deu um passo ousado para transformar o navegador em uma camada de execução de inteligência artificial.

Nesta terça-feira (30), a empresa anunciou o MCP Connector para o Opera Neon, uma tecnologia baseada no Model Context Protocol (MCP).

Na prática, isso significa que IAs externas, como Chat GPT e Claude, agora podem “entrar” no seu navegador para realizar tarefas, em vez de você ter que levar os dados até elas.

Diferente das extensões comuns, essa integração permite que as IAs operem dentro da sua sessão ativa.

Elas podem ler o conteúdo das abas abertas, capturar telas, preencher formulários e até abrir novas abas de forma autônoma, tudo com o seu contexto em tempo real.

Um dos maiores gargalos atuais é a necessidade de copiar textos de um site para explicar algo ao chatbot.

A IA passa a ter acesso direto ao ambiente onde o trabalho acontece.

“Conectamos clientes de IA populares diretamente a um navegador agêntico, permitindo que operem onde os usuários já trabalham”, explica Monika Kurczyńska, Diretora de P&D para IA da Opera.

A novidade suporta diversos agentes de peso, incluindo: Para garantir que essa “liberdade” das IAs seja segura, a Opera implementou um sistema de autenticação via URL segura e uma camada de proxy que monitora a conexão.

O objetivo é que a IA aja como um assistente que entende exatamente o que você está vendo na tela, mas sob seu controle.

O recurso já está disponível para os assinantes do Opera Neon.
