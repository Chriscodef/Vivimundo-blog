Software Microsoft corrige bug que fazia a busca do Windows 11 parar de funcionar Há 5 minutos

Por Kevin Ribeiro • 09/04/2026 às 16:10

A Microsoft corrigiu um problema no Windows 11 23 H2 que fazia a busca do Menu Iniciar exibir uma lista de resultados em branco.

O erro afetou uma pequena parcela de usuários desde 6 de abril deste ano.

A falha foi causada por uma atualização do Bing no lado do servidor voltada a melhorar a performance da busca.

Como efeito colateral, porém, usuários podiam se deparar com uma lista de resultados vazia, embora ainda clicável .

Nesses casos, nada aparecia como deveria na busca, nem mesmo aplicativos instalados localmente .

Alguns usuários também relataram que as buscas sequer eram realizadas.

A Microsoft não especificou quantos usuários foram afetados , mas destacou que o impacto foi limitado.

Embora o Menu Iniciar seja um dos principais recursos de navegação do Windows 11, ele está intrinsecamente ligado à internet.

O mecanismo de busca é integrado ao Bing , realizando consultas online a partir dos termos inseridos — o que faz com que falhas no servidor possam afetar diretamente o funcionamento do recurso.

Um incidente semelhante ocorreu em 2022, quando usuários do Windows 11 21 H2 sequer conseguiam iniciar o sistema após uma atualização .

Em 2023, outro bug do sistema operacional fez com que a busca e o Menu Iniciar parassem de responder.
